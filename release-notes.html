Beta 0.1
release notes.

The first public EdgeDock beta: reliable window parking, recoverable workflows, and a complete Windows control center.

Download Beta 0.1

Included

  • Dock windows on every monitor edge with a configurable modifier.
  • Native and glossy pull tabs with hover peek, movable anchors, and custom colors.
  • Stacks with live DWM previews, reordering, and transfers in Window Shuffle.
  • Named workspaces, per-application rules, remembered edges, and keyboard cycling.
  • Quick switcher, system tray, startup controls, settings import/export, and diagnostics.
  • Crash-safe journaling, emergency recovery, and display-layout resilience.
  • Reduced motion, high contrast, larger targets, tooltips, and onboarding.

Before testing

  • The executable is not code-signed yet, so Windows SmartScreen may warn.
  • Save important work and note your emergency recovery shortcut.
  • Administrator apps require EdgeDock to run at the same privilege level.
  • Unusual custom windows may not support standard docking APIs.

Requirements

Windows 10/11 · x64

No separate .NET installation is required. Unzip the download and launch EdgeDock.exe.
